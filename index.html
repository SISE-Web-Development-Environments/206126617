Questions and Answers

Mention and explain three main differences between web1, web2 and web3.
1.In web1 a web page is static - it cannot be changed nor modified by visitors (read only).
In web2 and web3 a web page can be dynamic, a visitor can comment or write something in the page.

2.In web1 the content of a web page is meaningless to computers and machines.
In web2 and web3 machines can "understand" the content of web pages by inputs from many users and integrate words into concepts (e.g. web search engines).

3.In web1 there is no interaction between a user and a machine. The user can open a web page and read its content only.
In web2 there is interaction between a user and a machine - any user can share information by creating or editing web pages as he wishes.
In web3 there is interaction between a user and a machine - a machine recommends or finds information for the user in accordance with the user's activity on the web.


Give an example for a use case that demonstrates the differences you have mentioned.
Let's demonstrate the differences related to a case when a user wants to find a recipe for a cake:

In web1 the user needs to enter a url of a website which is known for containing recipes (e.g. a chef's personal website) and find a recipe for a cake.

In web2 the user can search "cake recipe" in any search engine and get links to websites which contain cake recipes.

Also in web3 the user can search "cake recipe" in any search engine and get links to websites which contain cake recipes, but may also get links to websites which contain recipes for other pastries or foods, if they are similar to cakes.
